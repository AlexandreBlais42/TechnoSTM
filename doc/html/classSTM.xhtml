◆ StateMachineStates

typedef enum STM::StateMachineStates STM::StateMachineStates

États de la machine à états du microscope.

Note
Initialize : Monte le step moteur de 40 pour s'éloigner du matériel
Find_sample : Monte le matériel jusqu'à ce qu'il soit détecté ou que la hauteur maximale est atteinte
Lower_motor : Descend le matériel et ensuite le moteur
Mesure_height : Ajuste le microscope jusqu'à obtenir le voltage désiré
Save_pixel : Insère la hauteur mesurée dans l'image
Goto_next_coordinate : Déplace le microscope à la prochaine coordonnée pour l'image
